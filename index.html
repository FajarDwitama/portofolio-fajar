Proyek Akademik & Teknis

FTTH (Fiber to the Home) Network Design

Merancang infrastruktur dan topologi jaringan Fiber to the Home (FTTH). Proyek ini mencakup analisis perencanaan rute kabel, alokasi perangkat optik, dan pemodelan desain untuk memastikan distribusi layanan internet berkecepatan tinggi dapat berjalan secara efisien dan andal.

Prototipe IoT: Pendeteksi Kebocoran Gas & Api

Merancang dan membuat purwarupa sistem keamanan berbasis mikrokontroler dan sensor IoT. Alat ini berfungsi secara spesifik untuk mendeteksi indikasi kebocoran gas dan titik api lebih dini, sehingga dapat mencegah potensi kebakaran dan meningkatkan standar keselamatan lingkungan.
